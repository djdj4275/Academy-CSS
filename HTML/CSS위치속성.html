디스플레이 속성

블록속성 : P태그는 기본적으로 마진값을 가지고있다.

인라인속성 : width와 height값을 가질수없다.
div는 기본적으로 블록속성을 가지고 있음
display의 inline-block을 사용하면 width와 height를 가지는 inline처럼 쓸수있다.

Visiblity 속성확인

기본속성은 visible

hidden으로 숨김

none으로 없앰

Opacity 속성확인

opacity로 투명도 조절 : 0에 가까울수록 투명

Position의 속성확인

정적위치 : position의 기본값
상대위치 : 원래 가진 정적위치의 값에서 이동한 위치
고정위치 : 브라우저 화면 상에서 고정된 위치
절대위치 : 값이 고정된 top과 left에서는 위치가 고정
브라우저의 크기에 따라서 bottom과 right는 위치가 변형
